2016 | iOS: Swift 3.0 | Interface Builder | Realm | Node.js | AWS

CardBoard is a Magic The Gathering card list building application with a backend. The project, in its current state, gets high res images from Amazon Web Services, and uses a Node.js REST service to serve user lists and profiles via a PSQL database. This app demo allows users to build lists choosing from thousands of magic cards.

2014 - 2015 | HTML 5 | JavaScript | p5.js | Web Audio API

This live music visualizer was built on weekends between January and April of 2015. It is driven by our college radio's online audio stream and designed to function as a digital art piece at North Carolina State University's Hunt Library. It was completed as part of the library's inaugural "Code+Art" competition, and is still in rotation today.
