2017 - Current | iOS: Swift 3.0

Slides for Instagram is an iOS application that helps users post panoramic images to Instagram by leveraging Instagram's new gallery/slides feature. The application was built to be visually simple, and to caplitalize on users' existing knowledge of image manipulation gestures like pinch and pan to crop slides. This application was built with artist Yung Jake, and his agency Tight Corp.

2016 - Current | iOS: Swift 3.0 | iOS: Messages Framework

Emoji Ink Messenger is an iMessages application that allows you to combine emoji to create unique Stickers for use in iMessage conversations. What you see in the video below is a beta build. In the demo you can see me creating and placing stickers using the application. This application is an ongoing project with artist Yung Jake, and his agency Tight Corp.

Summer 2016 | iOS: Swift 3.0 | Realm | Node.js | AWS

CardBoard is a Magic The Gathering card list building application with a backend. The project, in its current state, gets high res images from Amazon Web Services, and uses a Node.js REST service to serve user lists and profiles via a PSQL database. This app demo allows users to build lists choosing from thousands of magic cards. The project is part of a larger concept which would serve as a social platform for users to trade Magic The Gathering cards locally.

2014 - 2015 | HTML 5 | JavaScript | p5.js | Web Audio API

This live music visualizer was built with a small team on weekends between January and April of 2015. It is driven by our college radio station's online audio stream and designed to function as a digital art piece at North Carolina State University's Hunt Library. It was completed as part of the library's inaugural "Code+Art" competition, and is still in rotation today.
